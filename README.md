## Features

- Clean Architecture + BLoC state management
- Responsive UI across
  - Desktop
  - Android
  - ios


## Running the App

### Android
To run the app on an Android device or emulator, use the following command:

```sh
flutter run
```

### Web
To run the app on Desktop, use the following command:

```sh
flutter build web
```
